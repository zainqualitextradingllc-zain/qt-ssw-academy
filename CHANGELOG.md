## [0.1.0] — 2025 (Prototype / プロトタイプ)

### Added / 追加

- **ダッシュボード / Dashboard**: コース進捗レーダーチャート・バーチャート・統計カード
- **JLPT N4 語彙 / Vocabulary**: 自動車整備専門語彙フラッシュカード 20語（習得トラッキング付き）
- **クイズエンジン / Quiz Engine**: 特定技能1号（自動車整備）試験対策クイズ 10問
- **QT InspectTag™ デモ / Demo**: NFCスキャンシミュレーション・点検レポート自動生成
- **証明書発行 / Certificate**: TECH READS®-NECH 修了証明書のプレビュー生成
- **言語切り替え / Language Toggle**: 日本語 / English 切り替え機能
- **商標表記 / Trademark Notices**: フッターおよびLEGAL.mdに完全な法的表記を掲載

### Tech Stack / 技術スタック

- React 18 + TypeScript
- Vite
- Tailwind CSS v4
- Recharts
- Noto Sans JP + Rajdhani (Google Fonts)
